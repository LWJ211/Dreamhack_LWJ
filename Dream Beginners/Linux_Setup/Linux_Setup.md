
# [Environment] WSL2 우분투 리눅스 환경 구축 및 실행 가이드

## 1. 개요
* **목적**: 드림핵 워게임 문제 풀이 및 Linux 환경 실습용 베이스 캠프 구축
* **OS 버전**: Ubuntu 24.04.4 LTS (Noble Numbat)

## 2. 설치 및 설정 과정
1. **관리자 권한으로 PowerShell 실행**
   * 단축키 `Windows + R`을 눌러 실행 창을 열고 `PowerShell`을 입력한다.
   * 관리자 권한 실행을 위해 `Ctrl + Shift + Enter`를 입력하여 파워셸을 구동한다.

2. **WSL2 우분투 설치**
   * 터미널 창에 아래 명령어를 입력하여 최신 LTS 버전인 우분투 24.04를 다운로드 및 설치한다.
   ```bash
   wsl.exe --install -d Ubuntu-24.04


* 설치 완료 메시지(`Changes will not be effective until the system is rebooted`)가 출력되면 컴퓨터를 **재부팅**한다.

3. **초기 유저 계정 및 패스워드 설정**
* 재부팅 후 아래 명령어로 설치를 마무리하고 인스턴스를 실행한다.


```bash
wsl.exe -d Ubuntu-24.04

```


* 기본 Unix 유저 계정(`user`)을 생성한다.
* 리눅스 터미널은 패스워드 입력 시 보안상 화면에 글자나 기호(`*`)가 출력되지 않으므로, 유의하여 입력을 완료한다. (`passwd: password updated successfully`)


4. **설치 버전 검증**
* 우분투 터미널 접속 후 버전을 체크하여 OS가 정상 구동됨을 최종 확인한다.


```bash
lsb_release -a

```



## 3. 우분투 실행 방법 (2가지)

설치가 완료된 후, 리눅스 환경에 진입하는 방법은 두 가지가 있다.

* **방법 1) Windows 검색 활용**
* 윈도우 하단 검색 창에 `Ubuntu`를 검색하여 설치된 앱을 직접 실행한다.


* **방법 2) PowerShell 명령어 활용**
* PowerShell을 열고 아래 명령어를 입력하면 우분투의 `home` 디렉토리(`~`)로 즉시 진입하여 작업할 수 있다.


```bash
wsl ~

```



## 4. 깨달은 점

* 무거운 가상머신 프로그램(VMware 등) 대신 윈도우 하위 시스템(WSL2)을 활용해 램 리소스를 아끼고 가볍게 리눅스를 띄우는 효율적인 방식을 배웠다.
* 리눅스 패스워드 입력 창의 숨김 처리 방식과 같이 CLI(Command Line Interface) 환경의 기본적인 보안 메커니즘을 경험했다.
* 구축 완료 후 `wsl ~` 명령어 등을 통해 윈도우 환경과 리눅스 환경을 유연하게 넘나들며 실행하는 접근 방식을 이해했다.
