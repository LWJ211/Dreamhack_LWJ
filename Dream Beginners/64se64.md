# [Dreamhack] Welcome - Web Hacking

## 1. 문제 개요
* **문제 설명**: 웹 페이지 소스 코드를 분석하여 숨겨진 플래그를 획득하는 문제
* **카테고리**: Web Hacking / Information Disclosure (정보 노출)

## 2. 풀이 과정
1. 웹 페이지 접속 후 소스 보기(`Ctrl + U`)를 수행하여 HTML 코드를 확인했다.
2. `<input type="hidden" name="64se64_encoding" value="...">` 태그 내부에 숨겨진 문자열 데이터를 발견했다.
3. 변수명을 기반으로 해당 문자열이 **Base64**로 인코딩된 것임을 식별하고 디코딩을 진행했다.
4. 디코딩 결과 출력된 파이썬 스크립트(아스키코드 배열을 문자로 치환하는 코드)를 실행하여 최종 플래그(`DH{...}`)를 획득했다.

## 3. 깨달은 점
## 3. 깨달은 점
1. 웹 소스 코드 내 `hidden` 태그에 중요 데이터를 방치하는 실수가 실제 모의해킹에서도 자주 발생하는 취약점임을 알게 되었다.
2. 이론으로만 접했던 Base64 인코딩과 아스키코드 변환을 실습을 통해 직접 해독해 보며 로우 레벨 데이터 표현 방식을 이해했다.
